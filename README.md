# Carlisle Technology

Webflow support assets for the Carlisle Technology site.

## Scripts

- `webflow/carlisle-technology/scripts/content/toc.js` builds a table of contents from headings inside `#single-article` and renders links into `#toc`.

After this repo is pushed, the latest-file jsDelivr URL is:

```html
<script src="https://cdn.jsdelivr.net/gh/blackpixelca/carlisletechnology@main/webflow/carlisle-technology/scripts/content/toc.js" defer></script>
```
